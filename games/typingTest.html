Typing Test
Information: This is a comventional typing game to test your typing speed and accuracy. A random paragraph is generated with some unconventional words for you to type as fast and as accurately as possible.
How To Play: Start typing to start the timer. The underlined word is the currect character. The characters will go blue if typed correctly and red if incorrectly typed. You can use backspace to correct your mistakes and improve accuracy. WPM is calculated based on speed and accuracy, and is used for the leaderboard. Type as much as you can in 30 seconds. You are not expected to finish in this time.
